# System Hub Dashboard

A private creator workflow dashboard for video planning, production status, earning outlook, daily wellbeing signals, and evidence-based action recommendations.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Build

`npm run build`

The production output is generated in `dist/`.

## Current data storage

The application currently stores dashboard records in browser local storage. Cloud authentication and database sync should be connected before relying on the deployed version across devices.
